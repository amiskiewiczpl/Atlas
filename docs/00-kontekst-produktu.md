# ATLAS HQ — kontekst produktu

## Czym jest ATLAS HQ

ATLAS HQ to prywatna aplikacja webowa tylko dla jednego użytkownika.

Nie jest to:

- zwykły task manager,
- Notion,
- Trello,
- Todoist,
- CRM,
- Excel,
- lista zadań.

To jest osobiste centrum dowodzenia życiem.

Inspiracje:

- Football Manager,
- Civilization,
- NASA Mission Control,
- Power BI,
- dashboardy Product Managerów,
- RPG Quest Log,
- Linear,
- Raycast,
- Arc Browser.

## Główna obietnica

Rano otwieram jeden ekran i wiem:

- gdzie jestem,
- co jest najważniejsze,
- gdzie są problemy,
- jakie są KPI,
- co mam zrobić dzisiaj,
- czy moje życie ma momentum.

## Filozofia

ATLAS HQ to dashboard decyzji, nie dashboard informacji.

Każdy element musi pomagać odpowiedzieć:

> Co mam zrobić teraz albo dzisiaj?

Jeśli komponent nie wspiera decyzji, nie powinien być na ekranie głównym.

## Główna pętla produktu

```text
poranny briefing
→ 3 ruchy dnia
→ wykonanie
→ wieczorny debrief
→ aktualizacja systemu
→ weekly review
```

8. Discovery

## Najważniejszy moduł: Discovery

Discovery nie przechowuje pomysłów.

Discovery przechowuje problemy rynkowe:

- kto ma problem,
- jak często,
- jak mocno boli,
- czy ludzie płacą,
- jak dziś sobie radzą,
- jaka jest konkurencja,
- jaka jest przewaga użytkownika,
- jakie są dowody.

## Najważniejsze ryzyko produktu

Największe ryzyko to przeładowanie.

ATLAS HQ nie może stać się pięknym cmentarzem danych.

Pierwsza wersja musi dowieźć tylko jedno:

> Użytkownik rano wie, jak grać dzień.
